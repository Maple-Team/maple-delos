## minio 资源的访问
前置条件：
1. bucket的access policy 为public
2. 获取资源的bucket的文件路径

访问资源的url为：`http://minio:9000/bucketName/filePath`
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
